# Javascript Practice

This repository contains my Javascript practice code and tutorials, organized by chapters.

## Structure

The codebase is divided into various chapters covering different Javascript concepts:

- **JS REVISION**: General revision of Javascript concepts.
- **CHAPTER-1**: Introduction and Basics.
- **CHAPTER-2**: Variables and Data Types.
- **CHAPTER-3**: Operators and Expressions.
- **CHAPTER-4 STRINGS**: String manipulation.
- **CHAPTER-5 ARRAY**: Array methods and properties.
- **CHAPTER-8**: Functions and Loops.
- **CHAPTER-9**: DOM Manipulation.
- **CHAPTER - 9 BROWSER EVENTS**: Handling browser events.
- **CHAPTER-10-CALLBACKS**: Understanding callbacks.
- **CHAPTER-11-PROMISSES**: Working with Promises.
- **CHAPTER-11-PROMISE - API**: Promise APIs.
- **Clock**: A simple clock implementation.
- **OOP**: Object-Oriented Programming principles.
- **HTML , CSS and JS - 6**: integrated projects.

## How to Run

Navigate to each directory to find specific Javascript files. You can run them using Node.js or by linking them to an HTML file and opening it in a browser.

```bash
node filename.js
```
